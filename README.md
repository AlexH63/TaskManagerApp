# Task Managing Web App
This web application allows you to manage your daily routines into an easy-to-use to-do list.

You may add, delete, or modify existing task items.

## Steps on how to run and test:
1. Clone the repo
2. On the terminal, go to the TaskManagerApp/server directory
3. type node server.js
4. Go to the TaskManagerApp/client directory
5. type npm start
6. Note that the web page should load and prompt the user for login
7. Select 'create an account'
8. Note that the Registration page appears prompting for a username, email, and password
9. Create an account and select register
10. Note that the user is brought back to the login page and prompts for the login
11. Log into the newly created account
12. Note that the Dashboard page titled "My Tasks" appears with entry fields prompting for a title and description
13. Enter any title and description and select 'Add'
14. Note that the task populates at the bottom
15. Select the 'Edit' button on the newly created task
16. Make any changes and click 'Save'
17. Verify that the changes were saved
18. Select the 'Delete' button.
19. Note that the task is removed from the list

## Resources utilized
JavaScript is the main programming language of this web application and Node.js/MongoDB is used for account creation and authentication as well as storing tasks.
React is used to display a flexible and user-friendly graphical user interface.
Node.js is used to handle HTTP requests through Express, which is a minimalist web app framework for Node.js.
